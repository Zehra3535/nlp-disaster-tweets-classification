# NLP Disaster Tweets Classification

## Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify whether a tweet is related to a real disaster or not.

The dataset comes from the Kaggle competition:
Natural Language Processing with Disaster Tweets.

## Problem
Social media platforms contain a huge amount of real-time information.  
Automatically detecting disaster-related tweets can help emergency services respond faster.

## Approach
The project follows a typical NLP pipeline:

1. Text cleaning and preprocessing
2. Tokenization and stopword removal
3. TF-IDF vectorization
4. Training machine learning models
5. Model evaluation

## Models Used
- Logistic Regression
- Random Forest (if you used it, keep it)

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK

## Result
The model successfully classifies disaster tweets with good accuracy and demonstrates the use of NLP techniques in real-world problems.
